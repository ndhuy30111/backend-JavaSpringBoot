# Đường dẫn Phương thức Method Params Chú ý
===Authentication===
Đăng nhập /api/Authentication Login POST { "userName" : " ", "password" : " " }
Lấy thông tin user /api/profile profile GET ghi chú : "Config Header" { headers:{ Authorization : "Bearer "+ Jwt } }
Đăng ký /api/dangky createUser POST { "userName" : " ", "password" : " ", "email" : " ", "name" : " " }
Danh sách User /api/account getUser GET Không có
===Category===
Danh sách category /api/category getAll GET Không có
Tìm category theo url /api/category findOneByUrl GET /api/category/{url}
Lưu thông tin category /api/category save POST {"name": " "} Cần đăng nhập
Xoá category /api/category/{id} save DELETE Không Cần đăng nhập
Sửa category /api/category/{id} save Put {"name": " "} Cần đăng nhập
===Product===
Danh sách category /api/getAll getAll GET ?page=0&size=50
Tìm product theo url /api/product findByUrl GET /api/product/{url}
Tìm color của Product theo id /api/{id}/product getProductColor GET /api/{id}/product
Lưu thông tin product /api/category save POST Được Params theo FormData "product": { name: "", price: 0, discount: 0, shortIntroduction: "", introduce: "", category: [], color: [ { name: "", code: "", size: [ { name: "", amount: 1, }, ], image: [{"fileName":" "//ten Hình ảnh được lưu}], //ten file thuoc color }, ], } "image":[]//chưa hình ảnh Cần đăng nhập
Xoá Product /api/product/{id} save DELETE Không Cần đăng nhập
Sửa Product /api/product/{id} save Put {"name": "", price: 0, discount: 0, shortIntroduction: "", introduce: ""} Cần đăng nhập
